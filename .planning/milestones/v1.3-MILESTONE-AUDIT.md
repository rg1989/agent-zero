### Integration Findings

**Finding 1 — Step 5 inline customization (Low)**
SKILL.md Step 5 has per-template inline hints for only 3 of 7 templates (flask-dashboard, flask-basic, static-html). The 4 new Phase 17 templates are covered in the "Template customization quick reference" section at the end of the document. Step 5 does not cross-reference this section.
*Affected:* SKILL-02
*Mitigation:* Quick reference at lines 251-286 covers all 7 templates comprehensively.

**Finding 2 — _GUIDE.md example name (Low)**
The "How to use a template" example in `_GUIDE.md` uses `my_app` (underscore), which conflicts with SKILL-03's hyphen-only naming rule. An agent copying this example verbatim would fail validation at Step 1.
*Affected:* SKILL-03
*Mitigation:* SKILL.md Step 1 validation command catches and rejects underscore names at runtime.

**Finding 3 — Regex display inconsistency (Info)**
SKILL.md Step 1 rule text shows `{1,28}` (min 3 chars) while the bash command uses `{0,28}` (min 2 chars, matching the prose). Pre-existing from Phase 16.
*Affected:* SKILL-03
*Mitigation:* Runnable command is correct; only displayed regex is wrong.
